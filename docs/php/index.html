mdg_add_rewrites()

mdg_add_rewrites(object $content) : object

URL rewriting

Rewrites do not happen for multisite installations or child themes

Rewrite: /wp-content/themes/themename/assets/css/ to /assets/css/ /wp-content/themes/themename/assets/js/ to /assets/js/ /wp-content/themes/themename/assets/img/ to /assets/img/ /wp-content/plugins/ to /plugins/

If you aren't using Apache, alternate configuration settings can be found in the docs.

Parameters

object $content

WordPress rewrite object.

Returns

object —

WordPress rewrite object with added rewrites.
