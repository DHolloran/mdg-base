Classes

MDG_WP_Admin Handles all of the customizations wp-admin.
MDG_Type_Page Handles anything custom for the default WordPress "page" post type
MDG_Images Handles adding custom image sizes and other global image related functionality.
MDG_Sidebar Determines whether or not to display the sidebar based on an array of conditional tags or page templates.
MDG_Type_Base This is a base for custom post type classes so they can all take advantage of the same logic for meta, transients etc.
MDG_Vcard_Widget Example vCard widget
MDG_Nav_Walker Cleaner walker for wp_nav_menu()
MDG_Type_Post Handles anything custom for the default WordPress "post" post type.
MDG_Walker_Comment Use Bootstrap's media object for listing comments
MDG_Meta_Helper Handles anything to do with custom meta.
MDG_Type_Stub This class can be used as a starting point to add new post types.

MDG_Generic Contains methods/properties that can be used by all classes.

mdg_add_rewrites()

mdg_add_rewrites(object $content) : object

URL rewriting

Rewrites do not happen for multisite installations or child themes

Rewrite: /wp-content/themes/themename/assets/css/ to /assets/css/ /wp-content/themes/themename/assets/js/ to /assets/js/ /wp-content/themes/themename/assets/img/ to /assets/img/ /wp-content/plugins/ to /plugins/

If you aren't using Apache, alternate configuration settings can be found in the docs.

Parameters

object $content

WordPress rewrite object.
